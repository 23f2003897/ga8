# Question 8: MLOps & DevOps Concepts Quiz

## Objective
Demonstrate an understanding of core MLOps and DevOps fundamentals, specifically regarding CI/CD pipelines, automation workflows, and Infrastructure as Code (IaC).

## Quiz Submission
**Answers Submitted: a,b,b**

### 1. Which file defines GitHub Actions workflows?
- **Selected Answer:** a (.github/workflows/*.yml)
- **Concept:** GitHub Actions looks for YAML configuration files strictly within the `.github/workflows/` directory to trigger CI/CD automation.

### 2. What does CI stand for in CI/CD?
- **Selected Answer:** b (Continuous Integration)
- **Concept:** The practice of merging code changes frequently into a central repository, where automated builds and tests (like Ruff linting) are run.

### 3. What does "IaC" stand for?
- **Selected Answer:** b (Infrastructure as Code)
- **Concept:** Managing and provisioning computing infrastructure through machine-readable definition files (like a `Dockerfile`), rather than physical hardware configuration or interactive configuration tools.
